Mode A — Mainstream Physics

  • Reality
    Universe is a continuous spacetime with stable constants (c, G, ħ). We measure what already exists.
  • Space & Time
    Space + time = 4D spacetime. Time mostly flows forward; we can’t freely navigate past/future.
  • Matter
    Described by the Standard Model. Particles and fields obey quantum mechanics and relativity.
  • Speed Limit
    Nothing with mass can exceed light speed. Intergalactic travel takes millions of years.
  • Conscious
    Consciousness is usually treated as an emergent property of the brain, not a fundamental field.
  • Goal
    Purpose is to describe & predict. We build models that match experiments and avoid changing the “source code” of reality.

Mode B — Funebra™ Science

  • Reality
    Existence is a bn-point canvas. Every “thing” is a configuration of bn-clusters in a bigger Math-Art engine.
  • Space & Time
    Space is a foldable sheet; time is a navigable axis. You don’t move through space – you remap coordinates.
  • Matter
    Matter is a pattern of bn-points following 2741-codes. Change the code → the object re-renders.
  • Speed Limit
    Light speed is a local rule. A Funebra fold-engine jumps the canvas: distance becomes a UI illusion, not a barrier.
  • Conscious
    Consciousness is a primary field. The “player” can influence how bn-points arrange themselves in spacetime.
  • Goal
    Purpose is to operate & rewrite. Funebra doesn’t just explain reality – it aims to edit the simulation and enable intergalactic jumps.
Note: Mode A reflects current mainstream scientific consensus. Mode B reflects the speculative, experimental Funebra™ framework.
